自訂樣式表 embedding 注意事項

  • 在兩個head標籤之間 用style標籤夾住 在寫其CSS程式碼
  • 選擇要修改的標籤做修改 例如:h1~h6 .......
  • 後面放上{ } 做修改 所有同標籤的都會被修改
  • 若剛好要指定的內容一樣 可以h1,h2{} 一起修改
  • class 選取方法 .類別名
  • class = "類別名"
  • 只讓特定標籤才能套用CLASS -- tag.類別名
  • 一個標籤可以有多個class 使用方法class="cl01 cl02 cl03"(用空格分開每個類別)
  • 上面的寫法 順序是沒差 但是上面的宣告順序有差 後的會覆蓋前的
  • 常用來一個類別 只宣告一兩項東西 例如color font-size 再讓標籤多拿幾個類別
  • id :使用# (class是.)
  • 同一個id名盡量只使用一次 因為JS一次只會做第一個找到的ID 若有重複的其他ID會出錯
  • 優先級別順序 style inline > id > class 同樣級別時 後的覆蓋前的
  • 引用樣式表 CSS檔
    • 需要有檔案 副檔名為.css
    • 將以下標籤放在兩個head標籤之中
    • link rel:"stylesheet" href ="檔案路徑和檔名"
    • rel:"stylesheet" 的意思即為樣式表 就是css
    • link 元素 和style 的先後順序 有差別 後的覆蓋前的 所以建議link放在style前面

邊框樣式

  • border 邊界的線 border : 1px solid green
  • 1px 是線的寬度 solid是線的型態(其餘還有dashed,double,ridge...) green是顏色
  • margin 外距 可以針對上下左右更改 若直接輸入4個 長度 將會依序是上右下左
  • padding 內距 可以針對上下左右更改 若直接輸入4個 長度 將會依序是上右下左
  • border-radius 邊界預設為方形 用此讓4角變圓滑 可以針對上下左右更改 若直接輸入4個 長度 將會依序左上 順時鐘旋轉
  • margin 可以設定成auto 就會左右自動置中 如果要離上面有高度 可以 80px auto

浮動

  • 浮動的寫法 是 float:left | right
  • float 後面的元素 都會受浮動的影響
  • 要取消浮動 可以使用clear : left | right | both

table 表格

  • table的基本屬性與child
    • 一個tr 是一個橫排
    • 一個tr 裡面放N個td或th 就會有N個格子
    • th td 差別在於一些預設的不同 (th會置中.粗體...)
    • tr:hover 可以讓滑鼠移至上面時 使同一列的表格產生變化(例如改變顏色)
  • 合併儲存格
    • colspan(橫向) rowspan(縱向) 可以合併儲存格
    • 被合併就要少一個 不然會凸出來 colspan 就是同一排的(tr)要少一格 rowspan 就是下一排的tr的對應位置要少一格儲存格
  • 群組化表格
    • thead tbody tfoot可以群組化表格 方便制定樣式
    • tbody tr:nth-child(2n){background-color: aquamarine;} 改變所有偶數列的寫法
    • text-align 是文字對齊 可以選置中.靠左.靠右 將群組表格統一調整 justify 左右對齊

表單

  • form 表單的屬性與屬性值
    • action 接收程式的url
    • method 傳送資料的方式 有get post
      • get 是用網址列將資料傳送出去 按送出後 可以看到網址的改變
      • post 則是用封包將資料傳送出去 所以如果接收的程式還沒寫 會跑不過
    • enctype 傳送時的編碼設定
  • input標籤中的屬性與屬性值
    • input 裡面的value屬性 是設定預設的字串
    • size 表示寬度
    • name 是用來分辨該資料是屬於哪一個 很重要 盡量設成 有意義的單詞
    • type屬性當中的 text表示文字輸入 password表示文字輸入時 會以*表示
    • input標籤中 type屬性當中的submit表示送出 reset是清除 預設已經設定了 不需要額外設定
    • maxlength:"N" 最大字元數=N (也會影響寬度)
    • checkbox 複選勾選框 value 表示回傳時所代表的意思
    • radio 單選勾選框 name相同 表示是同一題 同一個name最多只有一個勾選 value 表示回傳時所代表的意思
  • textarea標籤中的屬性與屬性值
    • cols 表示寬度 大概單位是字元
    • rows 表示高度 大概單位是字元
  • label 標籤中的屬性與屬性值
    • 是 inline 元素 直接修改寬度 會無效(因為inline元素大小為自己本身)
    • 但是配合浮動元素 可以讓寬度width重新有效
    • for:"XXXX" 其中XXXX可以對應到ID 使按下label時 可以直接到輸入文字模式 或是勾選方塊
  • 下拉式選單
    • select標籤的屬性與屬性值
    • name id multiple複選 沒有加就是單選
    • option標籤 value回傳值 selected預設選取 給使用者看的選項要寫在兩個option標籤之中
  • fieldset
    • fieldset標籤是用來將表單元件群組組件化
    • 常見修改項目:width,margin,border,background-color...等
    • legend 是群組的標題
  • 其他注意事項
    • 所有想要被傳送的資料都要放到form裡面
    • 原本按鈕預設是啟動狀態enable 可以改成disable不啟動狀態
    • button 是沒有預設功能的按鈕 需要自己用JS設定
    • button 有兩種寫法 一種是:input type="button" value="按鈕1". 另一種是:button type="button" 按鈕2 /button
    • 調整表單間距的建議作法 在每一個要區隔的地方加入div 然後對div做樣式設定

input type 屬性新類別

  • 日期時間選擇器: date,datetime-local,month,time,week...
  • 新文字輸入型方塊:email,url,tel,search...(傳送時瀏覽器會檢查,手機輸入時也有差)

  • 範圍驗證maxlength

  • multiple 加在input file 可以一次上傳多個檔案
  • disable 選不到 readonly(只有文字輸入有效)唯讀狀態,可以選但是編輯無法

  • ex: < input type ="range" name="number02" min="0" max="120" />
  • ex: < input type ="text" name="text01" pattern="[a-z,A-Z]{1}[1-2,8-9]{1}\d{8}" />

樣式屬性

  • display:
    • none 不顯示
    • block 以區塊的形式顯示 前後的元素都會換行顯示
    • inline 顯示 前後的元素會緊貼這個元素

  • visibilty
    • visble 顯示
    • inherit 如果父元素可見 就可見
    • hidden 隱藏 (但仍占有空間)(與display:none 不同)

    • auto 若超出容器 自動顯示卷軸 (最常用)
    • scroll 強制顯示卷軸
    • hidden 超出容器 就隱藏超出的部分
    • visible 全部顯示 無視區塊大小(預設)

css樣式屬性

  • 文字陰影 與區塊陰影
    • 寫法:text-shadow: 3px 5px 7px blue
    • 第一個屬性值 h-shadow 表示左右偏移的陰影 正為向右的距離
    • 第二個屬性值 v-shadow 表示上下偏移的陰影 正為向下的距離
    • 第三個屬性值 blur 是陰影的模糊的程度 越大越模糊
    • 第四個屬性值 color 為陰影的顏色
    • 區塊陰影寫法:box-shadow: 3px 5px 7px red; 四個屬性值同上
  • 定位(position:absolute|relative|static|fixed)
    • absolute 絕對位置 父元素內容區為邊界
    • relative 相對位置 原本應該在位置
    • static 靜態位置 依照原始碼的排列順序來定位 (預設)
    • fixed 固定位置 捲動卷軸時 不會移動
  • bottom|left|right|top : auto | length(X px) | percentage (X %)
  • 設定目標元素與邊界距離的屬性

RWD響應式網頁設計(Responsive Web Design) 與Media Queries

  • media types裝置類型
    • all:所有裝置
    • screen:螢幕裝置
    • print:印刷裝置，也包含預覽列印的畫面
    • speech:可以讀出頁面的設備
  • media feature裝置特性
    • width.height 螢幕的寬度與高度 可加上 min- . max-
    • orientation 螢幕旋轉方向 屬性值:portrait(直向) landscape(橫向)
    • aspect-ratio 螢幕長寬比 EX: 1/1 1920/1080
    • color 輸出螢幕的色彩位元數 若為0則代表黑白裝置
    • resolution 螢幕解析度
  • 有 or and not only 四種模式
  • 寫法範例:
    • @media screen and (min-width:400px) and (max-width:600px) { .....CSS設定.....}
    • <head> <link rel="stylesheet" media ="screen and (color)" href="example.css" /> </head>
    • @import url(example.css) screen and (color);
