JAVA 從request 和 session 取參數

從html端發送的form 請求
在JAVA servlet端 要獲取參數的語法如下:
request.getParameter("該變數的名稱");
這個方法拿出來的東西 必定是String
String 比較的時候 記得使用.equal()的語法
JAVA物件的比較必須這樣使用 ==是不能用的
如果是在form裡面 沒有填值 則會是空字串 也就是""
如果取的變數名稱不存在 則會回傳null
在比較null時 不可使用.equal() 要用==null

session的設定參數 使用的方法是
session.setAttribute("要取的變數名稱",某某東西)
取用參數的方法是session.getAttribute("變數名稱")
取出來的時候 是物件Object
要快速取用可以強制轉型
例如我存的是一個String的情況
就可以用
String myString=(String)session.getAttribute("我的變數名稱");
就可以直接取出來用了
