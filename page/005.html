POSTMAN常用功能與提醒

POSTMAN 是方便的測試後端的程式，也可以瞬間產生所需要的前後端的程式碼。 在URL區輸入想要測試的後台程式的網址，需要參數也可以放入。 如果是要模擬FORM表單的POST功能，因為有加密等等的情況， 要選擇的是BODY裡面的x-www-form-urlencoded， 這個才是實際使用POST的模式。

程式碼的選擇: HTML網頁建議是使用 jquery ajax JAVA 建議 OKHTTP (需要去載對應的JAR檔案，import之後才可以使用) NODEJS request 或AIXOS (也需要安裝對應檔案) python requests 以上只是筆者自己測試過，覺得很方便沒甚麼問題，實際上應該是每個也都可以。

瀏覽器會擋非同源的回應，但是POSTMAN可以正常測試，實際使用的時候記得要同源，或是用其他方法破解。
